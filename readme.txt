Accept Occomy payments on your WooCommerce store.

== Description ==

This is an Occomy checkout gateway for your WooCommerce store.

Occomy allows your customers to securely pay for purchases using their Occomy account. This results in higher store conversion rates and reduced fees. 

All Occomy payments are processed free of charge.

== Frequently Asked Questions ==

= Where do I get my API key? =

You can get your API key by navigating to [Occomy](https://www.occomy.com) and signing in using your Occomy account. Once signed in you will see a navigation toolbar on the left with an option marked 'API key' at the bottom. Please copy this API key and paste it under the 'API key' field in the WooCommerce payment settings for Occomy.
